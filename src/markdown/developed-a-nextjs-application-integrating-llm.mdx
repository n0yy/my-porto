### 🛠️ HLab AI | Personal Project

🗓️ **August 2024 - September 2024**

### 🌐 Overview
I developed **HLab**, a cutting-edge web application built with **Next.js**, leveraging **Gemini** LLM to provide advanced AI-powered features for diverse use cases:

- 📜 **Paper Summarization:** Simplify academic papers into digestible insights.
- 🛍️ **Product Description Generation:** Generate compelling product descriptions from minimal inputs.
- 🎓 **Lecture Summarization:** Extract key points and summaries from lecture materials.
- 📝 **CV Roasting:** Enhance resumes by providing actionable feedback for improvements.
- 📝 **Cover Letter Generator:** Generate a cover letter based on CV and Job Description will you apply
